# Compilation
School project (2020-2021) developed using C.

## Executing the project
### Part 1. Building the project
Use the MAKEFILE to build the project.
```bash
    $ cd Compilation/src
    $ make clean
    $ make release
``` 
### Part 2. Running the built program
Execute the C compiled object.
```bash
    $ cd Compilation/src
    $ ./program
```
